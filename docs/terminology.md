# 产品名词术语

## Serverless Framework

- service: 函数服务
- service instance: 函数实例

## Serverless Tencent (Components)

- app: 组件应用
- component instance: 组件实例

## 其他名词翻译

- auto-scalling: 弹性伸缩
- commands: 命令
- components: 组件
- dev-mode: 远程开发模式
- region: 地区
- registry: 注册中心
- stage: 环境
- serverless console: (腾讯云)应用中心
- template: 模板
